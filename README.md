# 🎧 Spotify Clone

A simple Spotify-inspired music player built as part of my Web Development course.
This project focuses on replicating the basic UI and functionality of Spotify using core web technologies.

---

## 🚀 Features

* 🎵 Music player interface
* ⏯️ Play / Pause functionality
* 📂 Song list display
* 🎨 Clean and responsive UI
* ⚡ Basic interactivity using JavaScript

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
